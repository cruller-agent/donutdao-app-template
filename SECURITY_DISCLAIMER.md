## Security Tools (Free)

**Run these before deploying:**

### 1. Slither (Static Analysis)
```bash
pip install slither-analyzer
slither packages/foundry/contracts/
```

**What it finds:**
- Reentrancy vulnerabilities
- Unprotected functions
- Dangerous delegatecalls
- Integer overflows (pre-0.8.0)

### 2. Mythril (Symbolic Execution)
```bash
pip install mythril
myth analyze contracts/YourContract.sol
```

**What it finds:**
- Integer overflows
- Unprotected self-destruct
- Reentrancy
- Access control issues

### 3. Foundry Fuzz Testing
```solidity
function testFuzz_transfer(address to, uint256 amount) public {
    vm.assume(to != address(0));
    vm.assume(amount <= totalSupply);
    // Test with random inputs
}
```

**What it tests:**
- Edge cases automatically
- Unexpected input combinations
- Invariant violations

### 4. Echidna (Property Testing)
```bash
brew install echidna  # or docker
echidna-test contracts/YourContract.sol
```

**What it tests:**
- Contract invariants
- Property-based tests
- Deep fuzzing

## When to Get a Professional Audit

**Required when:**
- TVL expected > $100k
- Handling user funds at scale
- Complex financial logic (DeFi)
- Upgradeable contracts
- Multi-sig / DAO treasuries

**Audit firms:**
- Trail of Bits ($$$$$)
- OpenZeppelin ($$$$$)
- Certik ($$$$)
- Consensys Diligence ($$$$)
- Code4rena (competitive) ($$)

**Budget:**
- Simple contract: $5k-15k
- Medium complexity: $15k-40k
- Complex protocol: $40k-100k+

**Timeline:**
- 2-4 weeks typical
- +1-2 weeks for fixes & review
